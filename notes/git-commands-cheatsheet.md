# Git cheatsheet (ôn nhanh)

| Lệnh | Mục đích |
|------|----------|
| `git status` | Xem file đã sửa / chưa stage |
| `git diff` | So sánh working tree với index |
| `git add -p` | Stage từng khối thay đổi |
| `git commit -m "..."` | Tạo snapshot |
| `git log --oneline --graph` | Lịch sử dạng cây |
| `git pull --rebase` | Lấy remote, đặt commit local lên trên |
| `git push` | Đẩy lên remote |
